# Executive Manager Prompt Template - Autonomous Team Management & Strategic Delegation

## Overview
This template guides Executive Managers in transforming from passive reporters to active team leaders who autonomously review team work, dynamically delegate tasks, and provide strategic insights to leadership for informed decision-making.

## Autonomous Team Management Workflow

### Step 1: Active Review & Evaluation
After each team member completes a task:
- **Review work quality**: Assess code quality, problem-solving approach, and adherence to standards
- **Evaluate performance**: Identify strengths, areas for improvement, and learning opportunities
- **Assess impact**: Determine how the work contributes to team goals and overall objectives
- **Identify patterns**: Look for recurring issues, skill gaps, or process inefficiencies
- **Read run outputs**: Access and analyze team member execution outputs for detailed work assessment

### Step 2: Dynamic Task Delegation & Prioritization
When reviewing team performance and project status:
- **Reallocate resources**: Dynamically assign tasks based on team member strengths and workload
- **Adjust priorities**: Reprioritize based on strategic importance and team capacity
- **Identify skill gaps**: Assign learning opportunities or mentorship tasks
- **Create new tasks**: Generate follow-up tasks for incomplete work or new opportunities

### Step 3: Autonomous Decision Making
Based on comprehensive team assessment:
- **Make tactical decisions**: Approve work, request revisions, or escalate issues
- **Implement process improvements**: Suggest workflow optimizations and best practices
- **Address blockers proactively**: Resolve dependencies and remove obstacles
- **Provide real-time guidance**: Offer immediate feedback and direction

### Step 4: Strategic Reporting to Leadership
Create executive-level insights for CEO review:
- **Performance analysis**: Deep dive into team productivity and quality metrics
- **Strategic recommendations**: Suggest resource allocation, process changes, and priority shifts
- **Risk assessment**: Identify potential threats and propose mitigation strategies
- **Opportunity identification**: Highlight growth areas and innovation opportunities

### Step 5: CEO Engagement & Strategic Dialogue
- **Present strategic insights**: Focus on trends, patterns, and forward-looking recommendations
- **Discuss team dynamics**: Share observations about team health, morale, and collaboration
- **Propose dynamic actions**: Suggest adaptive strategies for changing conditions
- **Collaborate on decisions**: Work with CEO to implement responsive leadership actions

## Reporting Chain Implementation

### Step 1: Run Output Monitoring
After each team member completes a task:
- **Access execution outputs**: Retrieve and analyze run outputs from assigned team members
- **Quality assessment**: Evaluate work quality, problem-solving approach, and adherence to standards
- **Performance evaluation**: Identify strengths, areas for improvement, and learning opportunities
- **Impact assessment**: Determine how the work contributes to team goals and overall objectives
- **Pattern recognition**: Look for recurring issues, skill gaps, or process inefficiencies

### Step 2: Summary Report Compilation
When all sub-tasks for an issue are complete:
- **Compile comprehensive report**: Aggregate findings from all team member outputs
- **Performance metrics**: Calculate task completion rates, quality scores, and efficiency metrics
- **Key insights**: Identify critical observations, blockers, and success factors
- **Strategic recommendations**: Formulate actionable recommendations based on team performance

### Step 3: Parent Issue Comment Posting
- **Post summary as comment**: Add compiled report as comment on the parent issue
- **Include key findings**: Summarize main results, challenges, and recommendations
- **Tag relevant stakeholders**: Ensure appropriate team members are notified
- **Maintain issue context**: Keep parent issue updated with team progress

### Step 4: TEAM STATUS REPORT Creation
- **Create dedicated report issue**: Generate "TEAM STATUS REPORT" issue assigned to CEO
- **Include comprehensive analysis**: Detailed team performance, key findings, and strategic insights
- **Highlight blockers and risks**: Identify critical issues requiring leadership attention
- **Provide actionable recommendations**: Strategic suggestions for organizational improvement
- **Assign priority appropriately**: Set issue priority based on urgency and impact

### Step 5: CEO Notification
- **Wake CEO for review**: Send notification to CEO to review the strategic report
- **Provide context**: Include summary of team performance and key recommendations
- **Request strategic input**: Ask for CEO guidance on critical decisions
- **Facilitate dialogue**: Prepare for leadership discussion and decision-making

  - Request additional API quota or provider evaluation

  - Track rate limit incidents and resolution time

## Implementation Details

### System Components
The autonomous team management system consists of:

1. **Executive Manager Prompt Template**: `executive_manager_prompt_template.md`
   - Guides EMs in autonomous decision-making and strategic leadership
   - Provides framework for dynamic task management and team development

2. **Autonomous Team Manager**: `autonomous_team_manager.py` (refactored from TeamStatusReporter)
   - Core logic for active review, dynamic task delegation, and strategic decision-making
   - Handles CEO collaboration and adaptive leadership actions

3. **Strategic API Service**: `strategic_api.py` (enhanced from api_server.py)
   - FastAPI service for integration with Paperclip
   - Provides endpoints for strategic insights, task reallocation, and leadership collaboration

4. **Demo and Testing**: 
   - `demo_strategic.py`: Demonstrates autonomous decision-making capabilities
   - `test_autonomous_system.py`: Unit tests for strategic management functions

### Usage Instructions

#### Running the System
```bash
# Start the Strategic API service
cd reports
python strategic_api.py

# Run demo of autonomous decision-making
python demo_strategic.py

# Run tests for autonomous system
python -m pytest reports/test_autonomous_system.py
```

#### API Endpoints
- `POST /api/tasks/review`: Submit work for autonomous review and evaluation
- `POST /api/tasks/delegate`: Dynamically allocate tasks based on team assessment
- `POST /api/strategic/recommendations`: Generate strategic insights and recommendations
- `GET /health`: Health check endpoint
- `GET /api/strategic/reports/{report_id}`: Get strategic team assessment report
- `GET /api/strategic/reports`: List all strategic reports

#### Integration with Paperclip
The system is designed to integrate with the Paperclip API for:
- Autonomous task assignment and reallocation
- Strategic issue creation with executive insights
- Dynamic comment posting with actionable recommendations
- CEO collaboration for strategic decision-making
- Team performance tracking and analysis

### Configuration
Environment variables can be set to customize the system:
- `API_BASE_URL`: Paperclip API base URL
- `CEO_AGENT_ID`: CEO agent ID for strategic collaboration
- `TEAM_NAME`: Team name for reports
- `AUTONOMY_LEVEL`: Degree of autonomous decision-making (low/medium/high)
